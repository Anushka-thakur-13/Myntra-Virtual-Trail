Myntra Virtual Trial: Guidelines

Get the best out of Myntra Virtual Trial with these guidelines.

Profile Image

  • Use a half-body image of yourself in a normal pose.
  • Ensure the image contains only one person.
  • For best results, wear top tanks or short sleeve clothes in your profile image. This allows for trying on various upper wear items effectively.

Source Image

  • Choose an image that clearly shows the clothing item you want to try on.
  • Ideally, the source image should feature a person wearing the clothing item.
